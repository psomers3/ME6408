float Car::get_hitch_angle ( )

Returns the desired hitch angle converted from the user input in the UserInput member.

This function will take the percentage of steering input obtained from the user and convert it to a percentage of the Min and Max radius defined in the Car.hpp file. These radii are based on physical limitations of the vehicle system.
